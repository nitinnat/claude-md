## What is Marp?

*   **Markdown Presentation Ecosystem:** A powerful tool for creating presentations with Markdown.
*   **Simple & Efficient:** Focus on content, not formatting headaches.
*   **Cross-Platform:** Works on Windows, macOS, and Linux.
*   **Open Source:**  Free to use and contribute.
*   **Flexible Output:** Generate PDFs, HTML slideshows, and more.

---

## Why Use Marp?

*   **Fast Development:** Write presentations quickly using familiar Markdown.
*   **Version Control Friendly:**  Track changes easily using Git.
*   **Clean Syntax:**  Easy to read and maintain.
*   **Customizable:** Control the look and feel with themes and CSS.
*   **Focus on Content:**  Let Marp handle the presentation layout.

---

## Basic Syntax & Structure

*   **Headers:** Use `#` for titles, `##` for subtitles, etc.
*   **Lists:** Use `*` or `-` for bullet points.
*   **Images:**  Insert images with `![alt text](image.jpg)`.
*   **Code Blocks:** Use triple backticks ``` for code snippets.
*   **Slide Separators:**  `---` creates a new slide.
*   **Comments:** `<!-- comment -->` for hidden notes.
